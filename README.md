GORN
====

![Gorn Creature from Star Trek](http://upload.wikimedia.org/wikipedia/en/a/a0/StarTrek-Gorn.jpg)

Gorn is a simple C++11 OpenGL 2.0 multiplatform renderer

Its main goals are:

* simplicity
* performance
* reusability
* draw call batching
* multithreading
* support for Linux, Windows, Android and iOS
